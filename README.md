HR Analysis Dashboard (Power BI)

This project contains an interactive HR Analysis Dashboard built using Power BI to analyze employee attrition, demographics, job satisfaction, and department-wise trends. It helps HR teams understand workforce behaviour and make better, data-driven decisions.

📌 Features

Overall employee summary (Total, Active, Attrition, Attrition Rate)

Department-wise and age-wise employee distribution

Gender-wise attrition across all age groups

Education field vs attrition analysis

Job satisfaction levels by job roles

Fully interactive visuals built in Power BI


🛠 Tools Used

Power BI Desktop

Power Query

DAX

Excel/CSV HR Dataset


📂 Files Included

HR-Analysis-Dashboard.pbit – Power BI template

HR-Analysis-Dashboard.pdf – Dashboard preview

HR Data.xlsx / HR data.csv – Dataset


🎯 Objective

To identify key HR insights such as attrition patterns, employee satisfaction, and demographic trends to support decision-making and improve employee retention.
